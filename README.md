Fitness Club

Учебный проект, разработанный на платформе 1С:Предприятие 8 в рамках дипломной работы после прохождения курса по разработке 1С. Конфигурация предназначена для автоматизации деятельности спортивного зала.

В ходе реализации проекта были применены навыки разработки на платформе 1С:Предприятие 8, включая создание объектов конфигурации, разработку управляемых форм, написание запросов и реализацию бизнес-логики приложения.

В процессе разработки использовались:

Библиотека стандартных подсистем (БСП);
механизм присоединенных файлов;
управляемые формы;
система компоновки данных (СКД) для разработки отчетов;
язык запросов 1С;
регистры накопления и регистры сведений;
работа с объектной моделью 1С (справочники, документы, перечисления, регистры);
механизмы обмена данными и работы с файлами.
